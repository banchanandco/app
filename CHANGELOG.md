# 📋 변경 이력 (Changelog)

---

## v1.0.0 — 초기 릴리즈

### 핵심 기능
- Firebase 인증 (Google OAuth + 허용 사용자 목록)
- PWA 지원 (홈 화면 설치)

### 재고 관리
- 7개 카테고리 (소스·양념, 야채, 고기, 컨테이너, 소모품, 청소, 기타)
- 입고 / 재고 / 출고 탭
- 박스→개 단위 변환 (box/case/bag → 개/lb 등)
- 개당 용량 설정 (예: 간장 1box = 12개, 1Gal each)
- 주문요망 (잔여 15~40%) / Urgent (잔여 15% 이하) 자동 감지
- 같은 품목 재입고 시 수량 자동 합산

### 주문 관리
- 거래처 등록 및 아이템별 거래처 지정
- 전체 / 거래처별 탭
- 주문리스트 텍스트 복사
- 쇼핑리스트로 전송

### 쇼핑 리스트
- 체크박스 리스트 (완료 항목 아래로 이동)
- 주문리스트 탭 (주문관리에서 가져온 항목)
- 메모장 탭 (2칼럼, 5줄 미리보기)

### 원가계산
- PIN 잠금
- 재료 단가 관리

### 레시피
- PIN 잠금
- 원가 자동계산
- 카테고리 분류 및 필터
- 재료 인라인 수정
- 1인분 기준 조절
- 재고 반영 버튼 (레시피 사용량 차감)

### 통계
- 입고 / 재고 / 출고 통계
- 일간 / 주간 / 월간 보기

### 스케줄
- 월별 캘린더

### Google Sheets 연동
- 인보이스 사진 AI 분석 (Gemini)
- 거래처별 자동 분류
- 단가 자동 저장 및 불러오기
- 월별 합계 자동 계산

### UI/UX
- 모바일 우선 디자인
- 태블릿 / 랩탑 max-width 580px 대응
- 세이지 그린 컬러 테마
- 드로어 네비게이션 (전 페이지 통일)
